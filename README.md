# RAG Q/A Chatbot

A simple **Retrieval-Augmented Generation (RAG)** Question/Answer chatbot built using Python.

## Features

- Uses **LangChain** for building the conversational AI.
- Integrates **Google Generative AI** for natural language responses.
- Stores and retrieves embeddings with **ChromaDB** vector database.
- Supports question-answering over your custom documents or datasets.

## Libraries Used

- [langchain](https://pypi.org/project/langchain/)  
- [langchain_google_genai](https://pypi.org/project/langchain-google-genai/)  
- [langchain_chroma](https://pypi.org/project/langchain-chroma/)
